About Me

Hello! I'm a computer vision researcher specializing in camera calibration techniques and Neural Radiance Fields (NeRF). My work focuses on developing accurate camera models for 3D reconstruction and novel view synthesis applications.

With a background in computer science and mathematics, I combine theoretical knowledge with practical implementation to advance the state-of-the-art in 3D scene representation and rendering. I'm particularly interested in improving the efficiency and accuracy of NeRF models for real-world applications.

Projects

Probability of Boundary Edge Detection

This project implemented a novel method for detecting boundary edges by estimating the probability that a given pixel is an edge. Based on the work Contour Detection and Hierarchical Image Segmentation by Pablo Arbelaez et al. Uses a large filter bank to create texture, brightness, and color maps, combining their gradients with Canny and Sobel baselines.

GitHub Documentation

Auto Panogram: Classical & DL Panoramic Image Stitching

Implemented a classical panoramic image stitching pipeline using corner feature matching, Asynchronous Non-Maximum Suppression (ANMS), and RANSAC outlier rejection techniques. Also implemented a deep learning-based pipeline using Deep Image Homography Estimation (DeTone et al.) with both supervised and unsupervised approaches.

GitHub Documentation

CIFAR-10 Image Classification (ResNet, ResNeXt, DenseNet)

Implemented and compared the performance of ResNet, ResNeXt, and DenseNet on the CIFAR-10 dataset. These models were all custom-built using only PyTorch layers, with no built-in implementations, providing deep insights into neural network architecture design.

GitHub Results

LQR and PD Controllers for 3D Quadrotor Trajectory Tracking

Designed an optimal LQR controller for a 6-DOF quadrotor using hover state linearization, reducing the non-linear trajectory tracking error by 82% for quadrotor stabilization. Implemented a PD control system to achieve precise hovering and trajectory tracking for non-linear paths. Validated the PD hover controller's performance through tests at random hovering positions and headings.

GitHub Demo

Forward and Inverse Kinematics for OpenManipulatorX Using ROS 2

Designed and implemented forward and inverse kinematics algorithms to control the OpenManipulatorX robotic arm using ROS 2. Utilized ROS 2 to develop publisher-subscriber and client-server nodes, enabling real-time motion control and seamless integration of kinematics computations. Achieved precise robotic movements for predefined positions and successfully performed object manipulation tasks.

GitHub Documentation

Multi-Camera Calibration Toolkit

An open-source toolkit for calibrating multi-camera systems with high precision. Includes tools for intrinsic and extrinsic calibration, distortion correction, and validation. Implemented in Python with OpenCV.

Fast-NeRF Implementation

Developed a Classical Structure from Motion (SfM) pipeline to reconstruct 3D structures from sequences of 2D images. This project integrates key techniques such as feature detection, matching, motion recovery, and 3D reconstruction. Utilized essential matrix computation, bundle adjustment, and triangulation methods to accurately estimate 3D points and camera positions, demonstrating the core principles of SfM in computer vision.

GitHub Demo

Self-Calibrating SLAM System

A Simultaneous Localization and Mapping (SLAM) system that incorporates online camera self-calibration. Allows for robust operation even when camera parameters change during operation.

Research Focus

Camera Calibration

My research in camera calibration focuses on developing robust methods for estimating camera parameters in challenging environments. This includes work on:

  • Self-calibration from scene constraints
  • Multi-camera system calibration
  • Online calibration refinement
  • Handling non-standard camera models

Neural Radiance Fields

My work with NeRF technologies explores ways to make these powerful representations more practical for real-world applications:

  • Reducing training time through efficient sampling
  • Incorporating camera parameter uncertainty into NeRF models
  • Extending NeRF to handle dynamic scenes
  • Developing compact representations for resource-constrained devices
