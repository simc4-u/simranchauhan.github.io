Research & Projects
Classical Structure from Motion(SfM)
project page

Reconstructed a 3D scene from a collection of 2D images by implementing a Structure from Motion pipeline with advanced geometric optimization techniques. The approach combined triangulation methods with robust feature matching, PnP algorithms for precise camera pose estimation, and sparse bundle adjustment to refine both camera parameters and 3D points simultaneously. This pipeline enables developing complete 3D models from 2D images, creating accurate digital representations of real-world scenes with sub-pixel precision.
Camera Calibration
project page

A robust camera calibration system based on Zhang's method, using checkerboard patterns to extract camera intrinsic and extrinsic parameters. The approach employs optimization techniques to minimize geometric errors and models lens distortion to achieve high-precision calibration results. This calibration pipeline significantly reduces reprojection errors, enabling accurate 3D measurements and reliable computer vision applications.
Classical and DL based Panorama Image Stitching
project page

A panorama stitching system that combines multiple images into wide-angle views with seamless transitions. The approach uses feature detection, robust homography estimation, and advanced blending techniques to handle perspective differences between images. In parallel, deep learning models for homography estimation were implemented, including both supervised architectures and unsupervised approaches with photometric loss functions, each demonstrating superior performance compared to conventional methods.
Boundary Detection using PB-lite
project page

A multi-feature edge detection system that identifies object boundaries by analyzing texture, color, and intensity discontinuities across multiple scales. This implementation is based on a simplified version of Pablo Arbelaez's boundary detection algorithm, providing effective edge detection while being computationally more efficient than the original approach.
